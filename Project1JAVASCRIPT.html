Project 1

JavaScript

The interactive part of the website, other than hyperlinks, is JavaScript. Things such as buttons and calculators are created using JavaScript. Buttons can change the appearance of a website or submit information. Form submit buttons are created with html, but made effective with JavaScript. Simple calculators are made with more complex JavaScript than buttons, using if, else, and else if statements. Overall, JavaScript allows websites to become more interesting by being interactive and transmitting user comments and reviews to better the website.
