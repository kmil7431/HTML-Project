Proyecto 1

JavaScript

La parte interactiva del sitio web, aparte de los hipervínculos, es JavaScript. Las cosas tales como botones y calculadoras se crean con JavaScript. Los botones pueden cambiar la apariencia de un sitio web o enviar información. Los botones de envío de formulario se crean con html, pero se hacen efectivos con JavaScript. Las calculadoras simples están hechas con JavaScript más complejo que los botones, utilizando las instrucciones if, else y else if. En general, JavaScript permite que los sitios web sean más interesantes al ser interactivos y transmitir los comentarios y opiniones de los usuarios para mejorar el sitio web.
