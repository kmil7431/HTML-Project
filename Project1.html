The Structure of Websites

HTML is the code that creates the structural skeleton of websites. The things such as words and images are created using HTML. Without HTML, there couldn't be a website.

 The Creativity in Websites

CSS is the code used to design websites to look a certain way. It can be used to make the website beautiful, as well as show certain things. CSS is used to personalize websites

 The Interactive Part of Websites

JavaScript is used for the interactive portions of websites. Its uses are mostly to work with the person using the website and to relay information.
