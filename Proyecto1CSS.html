Proyecto 1

CSS

CSS se utiliza principalmente para embellecer sus sitios web mediante la manipulación de colores, diseños y fuentes. El control de color puede ser específico para un determinado elemento o para cada elemento de un determinado tipo. Además, los cambios que puede realizar en el color se extienden desde la propia fuente hasta las líneas de fondo y borde. Al igual que el control sobre el diseño que se muestra en HTML, CSS puede mover párrafos e imágenes a través de su sitio web. Finalmente, el control sobre la fuente incluye el tamaño de fuente, la alineación, el color y el tipo de fuente. En general, CSS se utiliza para permitirle personalizar su sitio web y permitir que la presentación de su sitio web sea profesional.
