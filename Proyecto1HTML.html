Proyecto 1

HTML

Además de crear texto e imágenes simples en su sitio web, HTML se puede usar para crear encabezados de carios tamaNos, crear estilos básico, organizar imágenes y texto dentro de la página web y vincylar diferentes sitios web con hipervínculos. Los distintos tamaNos de encabezados se pueden utilizar para mostrar distintos niveles de importancia dentro del contenido del sitio web. HTML puede crear estilos básicos que hacen que su sitio web parezca menos aburrido. La organización de imágenes y texto dentro del sitio web puede ser útil para garantizar que se consume menos espacio, para mostrar las relaciones entre las imágenes y los párrafos, y para que sy sitio web esté más ordenado. Finalmente, los hipervínculos pueden vincular sitios web no relacionados y ayudarlo amostrar sus fuentes, o acreditar ciertos sitios para que no sea acusado de plagio. En general, HTML crea el sitio web que se ve y se aprende.
