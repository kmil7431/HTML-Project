La Estructura de los Sitios Web

HTML es el código que crea el esqueleto estrutural de los sitios web. Las cosas tale como palabras e imágenes se crean utilizando HTML. Sin HTML, no podría haber un sitio web.

 La Creatividad en los Sitios Web

CSS es el código que se usa para diseñar sitios web para que se wea de cierta manera. Puede utilizarse para hacer wue el sitio web sea hermoso, así como para mostrar ciertas cosas. CSS se utiliza para personalizar sitios web.

 La Parte Interactive de los Sitios Web

JavaScript se utiliza para las partes interactivas de los sitios web. Sus usos son principalmente para trabajar con la persona que usa el sitio web y para transmitir información.
